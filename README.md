# xs-potato 🥔

A simple, mobile-first PWA for stock control and cash management, built for a family-owned produce shop.

<p align="center">
  <img src="src/assets/image.png" alt="App screenshot" width="320" />
</p>

## Features

- **Income & Expense tracking** with color-coded tabs (green / red)
- **Product autocomplete** from transaction history with unit price pre-fill
- **Inline typo correction** with bulk rename across all past records
- **History view** with Day / Month / Year filters and income/expense totals
- **Inline editing** — tap any value in the history to fix it on the spot
- **CSV export** of the currently filtered view
- **Installable PWA** with offline support

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- localStorage (no backend)
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
